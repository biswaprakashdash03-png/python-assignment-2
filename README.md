# 🐍 Python Practice Questions Collection

This repository contains beginner-level Python practice questions for learning programming fundamentals like conditions, loops, error handling, and basic calculations.

---

## 📌 Questions Included

### 1️⃣ Divisibility Check with Error Handling
Write a Python program that:
- Takes an integer input from the user
- Checks whether the number is divisible by 7
- Uses error handling (`try-except`) to handle invalid input
- If input is not valid, display an error message

---

### 2️⃣ Condition Checking Using OR Operator
Write a Python program that:
- Takes an integer input from the user
- Checks if the number is:
  - Greater than 70 OR
  - Divisible by 5
- Prints result as True or False based on condition

---

### 3️⃣ Factorial of 7 Using Loop
Write a Python program that:
- Calculates factorial of 7 using a loop
- Uses multiplication logic from 1 to 7
- Displays the final result

---

### 4️⃣ Volume of a Cube with Condition Check
Write a Python program that:
- Takes side length of a cube as input
- Calculates volume using formula (side × side × side)
- Prints the volume
- Checks if volume is greater than 1000 or not

---

### 5️⃣ Shopping Bill with Discount Calculation
Write a Python program that:
- Calculates total price of items:
  - Monitor, CPU, Mouse, Keyboard (fixed quantities and prices)
- Calculates total bill
- Applies discount:
  - 10% if total > 50000
  - Otherwise 5%
- Displays discount and final amount
